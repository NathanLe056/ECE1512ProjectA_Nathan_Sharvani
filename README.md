# ECE1512 Project A: State Space Models (Mamba) and Vision-Language Models (LLaVA)

## Overview
This repository contains code, and supplementary material for our ECE1512 Project A submission.
The project has two main components:
- **Part 1 & 2 (SSM)**: Analysis of the Mamba architecture and proposed extensions (MS-Mamba, AC-Mamba)
- **Part 3 & 4 (VLM)**: Analysis of LLaVA and proposed efficiency optimization via selective visual token reduction.
